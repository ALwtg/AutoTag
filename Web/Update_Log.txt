2026-01-07 更新日志 (V1.2.0)

[新增功能]
1. 统一标签管理系统
   - 支持上传自定义 `classes.txt` 文件，用于固定物体识别的类别 ID 标准（如 0=公共汽车, 1=自行车）。
   - 实现了自动标签扩展功能：当识别到 classes.txt 中不存在的新物体时，自动将其添加到类别列表中并分配新 ID。
   - 优化了 API 提示词策略，自动注入当前已知类别列表，提高识别一致性。
   - 批量导出（图片/视频）时，生成的标注文件将严格遵循统一的类别 ID 标准。

2. 全局 API 速率限制 (Rate Limiter)
   - 新增 API 请求速率控制功能（次/分钟），有效防止触发服务商的 Rate Limit 错误（如 429）。
   - 支持设置为 "-1" 以解除速率限制，全速运行。
   - 与并发控制功能协同工作：并发决定同时发起的任务数，速率限制决定任务放行的频率。
   - 界面优化：该设置项现已调整为常驻显示，不再随模式切换而隐藏。

3. 视频导出体验优化
   - “导出所有已标记视频”功能新增实时进度条反馈。
   - 显示总进度以及当前处理文件的具体百分比。

4. 动态配置加载
   - “选择模型”下拉菜单现在根据 `config.js` 中的配置项动态生成，无需修改 HTML 源码即可增删模型。

[优化与修复]
- 优化了批量处理时的标签映射逻辑。
- 调整了 UI 布局，使高级设置更加合理。
